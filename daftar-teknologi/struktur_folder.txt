STRUKTUR FOLDER - GAME PRESIDEN (HASIL KONSOLIDASI)
=================================================

Sesuai permintaan Anda, seluruh folder pendukung dimasukkan ke dalam folder `admin` atau `app` sehingga hanya ada 2 folder utama di root project:

/GAME-PRESIDEN
│
├── /admin (Manajemen & Infrastruktur)
│   ├── /frontend             # Next.js (Admin Dashboard)
│   ├── /backend              # Go (Admin API)
│   └── /infrastructure       # Folder Pindahan
│       ├── /database         # PostgreSQL Migrations & Scema
│       ├── /proto            # Definisi gRPC
│       ├── /redis            # Konfigurasi caching
│       └── /docker           # Dockerfiles & Compose
│
└── /app (Aplikasi Game & Core Logic)
    ├── /frontend-desktop     # Next.js + Tauri
    ├── /frontend-mobile      # React Native
    ├── /backend              # Go (Game Server)
    └── /shared-ai            # Folder Pindahan (Core Engine)
        ├── /python-ai        # Prototyping
        └── /cpp-core         # Penyempurnaan AI

=================================================

PENJELASAN:
1. **admin**: Kini merangkap sebagai pusat kendali infrastruktur (Database/Docker) dan dashboard editor.
2. **app**: Kini menjadi satu paket lengkap aplikasi game beserta "otak" AI-nya di dalam folder `shared-ai`.
3. **Hasil**: Root project terlihat sangat rapi hanya dengan 2 folder utama.
