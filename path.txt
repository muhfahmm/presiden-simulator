=== PRESIDEN SIMULATOR (EM2) ===

# Cara Cepat (Install & Jalankan)
cd "c:\fhm\em\app\frontend"
npm run setup
> Perintah di atas akan otomatis melakukan 'npm install' dan langsung menjalankan game.

# Cara Jalankan (Harian)
cd "c:\fhm\em\app\frontend"
npm run dev
> Akses: http://localhost:4000
> Log: NEXT (Cyan), GO (Magenta), AI-RUST (Green), AI-GO (Blue) akan tampil bersamaan.

# Lokasi Server Backend (Go)
Lokasi: c:\fhm\em\app\frontend\src\app\server\go
> Server ini berjalan otomatis saat 'npm run dev' dipanggil.

# Lokasi Konfigurasi Tauri (Desktop/Mobile)
Lokasi: c:\fhm\em\app\frontend\src-tauri\tauri.conf.json
> Gunakan file ini untuk mengubah Icon, Identifier (Bundle), atau Title Aplikasi.

# Cara Jalankan (Hanya Server / Backend)
Pilihan 1 (Lewat NPM):
cd "c:\fhm\em\app\frontend"
npm run dev:go

Pilihan 2 (Langsung via Go):
cd "c:\fhm\em\app\frontend\src\app\server\go"
go run main.go

# Cara Masuk (Mode Desktop - Tauri)
cd "c:\fhm\em\app\frontend"
npx tauri dev

# Cara Build (Produksi Desktop - EXE)
cd "c:\fhm\em\app\frontend"
npx tauri build
> Hasil (.exe): src-tauri/target/release/bundle/msi/

# Cara Masuk (Mode Mobile - Tauri Android)
Inisialisasi (Sekali saja):
cd "c:\fhm\em\app\frontend"
npx tauri android init

Jalankan:
cd "c:\fhm\em\app\frontend"
npx tauri android dev


# Tutup / Force Stop (Jika Macet)
Stop Game: taskkill /F /IM node.exe
Stop Server: taskkill /F /IM goserver.exe
Stop Semua: taskkill /F /IM main.exe

# Git Manajemen
git add .

git push origin main

# Teknologi yang Digunakan (Multi-Language)
Detail Lengkap: c:\fhm\em\app\frontend\project_tech_stack.json
1. Next.js (TypeScript) -> Frontend & UI (Label: NEXT)
2. Go -> Game Engine & Backend (Label: GO & AI-GO)
